## Main Concept

Generative AI (Gen-AI) is a subset of Deep Learning that is used to generate **new data** that is similar to the data it was trained on. So it is used to create content like text (conversations), images, code, music, video, etc...

## Key Aspect
- Generative AI use Generative Model, which in turn could be a [[20251029T060940-foundation-model|Foundation Model]] or a smaller specialized model.

## Basic Example
- You train the Generative Model with a lot of pictures of dogs, but you also train the model with a lot of pictures of cartoons, so the model is capable to identify what is a dog and what is a cartoon.
- Then the Generative Model is capable to **generate** a completely new image (never seen before by the model) that is a dog but looks like a cartoon.
