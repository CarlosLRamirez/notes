## What is Amazon Q Developer

- It's a GenAI tool provided by AWS that uses Foundation Models to provide developer productivity 🤔
- It generates real time code suggestion based on natural language comments and prior code (like autocompletion based on your code context)
- It works on an integrated development environment (IDE)
