# Setup Github Organization
## Concepto Principal
Las Organizaciones de Github son cuentas compartidas donde empresas y proyectos open-source pueden colaborar a travez de múltiples proyectos al mismo tiempo. Ofrece funcionalidad avanzadas de seguridad y administración.

La Organización de Github sirve como un contenedor para el trabajo compartido y le da al trabajo un único nombre. Cada usuario se logea con su cuenta personal, y varias personas puede colaborar en un proyecto compartido uniéndose a la misma cuenta de organizacion.
