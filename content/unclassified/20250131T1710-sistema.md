# Sistema
## Concepto Principal
Un sistema se puede definir como un **conjunto** de **elementos** que interactuan o tienen alguna  interdependencia entre sí  para lograr un **propósito común.**

## Contexto
Es importante comprender que es un sistema en el contexto del modelado del mismo y su simulación

## Aspectos Clave
- Un sistema es comúnmente afectado por cambios que ocurren fuera del sistema mismo, a esto se le conoce como el **entorno o ambiente del sistema**
- Cuando modelamos un sistema es importante definir las **fronteras** del sistema, es decir la donde separar el **sistema** de su **ambiente**, esta decisión depende del propósito del estudio.

## Aplicaciones
En todas las áreas de la vida existen sistemas
## Ejemplos
- Sistema de Transporte Público
- Sistema Digestivo
- Sistema Nacional Interconectado (SNI)
- Sistema de Sensores IoT
