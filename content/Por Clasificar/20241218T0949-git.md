# Git

## Definición
- Es el sistema de control de versiones mas utilizado hoy en día.
- Fue creado en 2025 por Linus Torvalds, específicamente para los desarrolladores del kernel Linux
- Es un  [Sistema de Control de Versiones Distribuidos](20241217T1507-sistemas-de-control-de-versiones-distribuidos.md) DVSC)
- Es gratuito y de código abierto (open-source).
