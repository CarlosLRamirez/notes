## Concepto Principal
La Inteligencia Artificial es una rama de las ciencias de la computación  la cual se dedica a resolver problemas que regularmente serian asociados con la inteligencia humana. 🤔

## Aspectos Clave
- La Inteligencia artificial involucra los siguientes elementos y procesos
	- Un Dataset de Entrenamiento (Training Dataset)
	- El proceso de entrenar un modelo de AI a partir del Data Set
	- Un científico de datos es quien entren el modelo
	- El usuario que utiliza el modelo previamente entrenado para obtener alguna respuesta del mismo.

## Ejemplos
- Un ejemplo básico de un modelo de AI podria ser un algoritmo de clasificación, el cual contiene una colección de imagenes de frutas clasificadas de acuerdo a la fruta que corresponde, el modelo fue previamente entrenado con una gran cantidad de imagenes de diferentes frutas.
- Un usuario puede introducir una imagen nueva de una fruta, y el modelo sería capaz de clasificar que tipo de fruta es, sin haber antes tenido esa imagen exacta. 


## Aplicaciones
Areas de aplicación de la IA:

	- Reconocimiento de imágenes
	- Conversión de voz a texto
	- Aprendizaje
